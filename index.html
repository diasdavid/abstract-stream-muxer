Intro

Installable via npm install --save interface-stream-muxer, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save interface-stream-muxer

You can then require() interface-stream-muxer as normal:

const interfaceStreamMuxer = require('interface-stream-muxer')

In the Browser

Interface-stream-muxer should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of interface-stream-muxer, including index.js and index.min.js, are included in the /dist folder. Interface-stream-muxer can also be found on unpkg.com under

  • https://unpkg.com/interface-stream-muxer/dist/index.min.js
  • https://unpkg.com/interface-stream-muxer/dist/index.js
